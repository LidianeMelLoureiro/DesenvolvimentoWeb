Desenvolvimento Web e o Front-End

Desenvolvimento Web

Criação de um Software que esteja conectado a uma rede internet ou intranet

O profissional dessa área tem como objetivo atender as nescessidades de seus clientes, desenvolvendo softwares completos, seja um site, aplicativo ou um portal. Esse profissional irá desenvolver habilidades técnicas em Linguagens de Programação, Linguagens de Marcação de Texto e também Folhas de Estilo para o melhor desenvolvimento de seu trabalho.

Front-End

Parte visual dos softwares

O front-end é toda a parte visual sendo todo o design, ferramentas de interação com o usuário e a interface de navegação, usados em sites, aplicativos, sistemas ou projetos. Resumindo é a parte que deve impressionar o cliente para o crescimento de seus négocios.
